All in One Place - Clusterização de Clientes

O objetivo deste projeto é fazer a clusterização dos melhores clientes da empresa All in One Place, para formar um programa de fidelidade chamado Insiders.
Em posse dessa lista, o time de Marketing fará uma sequência de ações personalizadas e exclusivas ao grupo, de modo a aumentar o faturamento e a frequência de compra.
Como resultado para esse projeto, é esperado que seja entregue uma lista de pessoas elegíveis a participar do programa Insiders e um relatório respondendo as seguintes perguntas:

  1. Quem são as pessoas elegíveis para participar do programa de Insiders?
  2. Quantos clinetes farão parte do grupo?
  3. Quais as principais características desses clientes?
  4. Qual a porcentagem de contribuição do faturamento, vinda do Insiders?
  5. Qual a expectativa de faturamento desse grupo para os próximos meses?
  6. Quais as condições para uma pessoa ser elegível ao Insiders?
  7. Quais as condições para uma pessoa ser removida do Insiders?
  8. Qual a garantia que o programa Insiders é melhor que o restante da base?
  9. Quais ações o time de marketing pode realizar para aumentar o faturamento?

Durante o planejamento da solução, durante a fase benchmark, eu me deparei com uma forma de clusterizar os clientes chamado modelo RFM (Recency, Frequency, Monetary, em inglês). Esse modelo permite fazer uma rápida entrega de valor a empresa, dando início ao programa de fidelidade. Com isso meu planejamento ficou da seguinte forma:

  • Começar aplicando RFM para gerar valor e trazer as principais características dos clientes;
  • Relatório de Insights;
  • Treinar algoritmos de ML(usar RFM para entrada);
  • Entrega da clusterização e relatório Final.

Como eu farei duas entregas, nesta página você encontrará os resultados das duas, começando pelo modelo RFM.

O que é o Modelo RFM?

RFM é uma ferramenta de marketing utilizada para clusterizar os clientes tendo como base três características:

  • Recency (Recência) – A recência nos aponta quando foi a última vez que o cliente fez uma compra;
  • Frequency (Frequência) – A frequência é a soma de todas as compras feitas pelo cliente dentro de um determinado período;
  • Monetary (Monetário) – Monetário é o total gasto pelo cliente dentro do período.

Com base nessas três características nós atribuímos notas que vão geralmente de 1 a 5, sendo 5 a nota mais alta. Para atribuir essas notas, devemos ordenar a característica de forma que os maiores valores fiquem no topo e os menores valores fiquem na base, após isso é atribuída a nota conforme a ordenação. Para os 20% que ficarão no topo da lista é atribuído a nota 5, e assim por diante até os últimos 20% que receberão a nota 1.
Com base na combinação destas notas nós podemos atribuir um rótulo ao cliente. O gráfico abaixo mostra os rótulos aplicados.
